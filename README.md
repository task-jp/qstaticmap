# qstaticmap

google static map like service based on [QtHttpServer](https://code.qt.io/cgit/qt-labs/qthttpserver.git/)

# requirement
- Qt 5.12 or later
- QtHttpServer latest

# build and run

```
$ qmake
$ make
$ ./qstaticmap -platform offscreen // or minimal on mac
```

# try
